# **📜 Історія Оновлень (Changelog)**

Усі помітні зміни у проєкті **FocusQuest** будуть документуватися в цьому файлі.

Формат базується на [Keep a Changelog](https://www.google.com/search?q=https://keepachangelog.com/en/1.0.0/), а версіонування відповідає етапам нашого [ROADMAP.md](https://www.google.com/search?q=./ROADMAP.md).

## **\[v1.5.0\] \- 2026-XX-XX (Фаза 5: Глибока кастомізація)**

### **Додано**

* **Класи персонажів:** Можливість вибрати шлях (Воїн, Маг, Розбійник). Класи дають пасивні бонуси (менші штрафи, більше XP або Золота).  
* **Система Досягнень (Achievements):** 6 унікальних бейджів за виконання особливих умов (наприклад, безперервний фокус \> 60 хв або купівля 5 зілль) із цінними нагородами.  
* **Оцінка часу (Estimates):** Поля для очікуваного часу на квести та кроки. Виконання завдання швидше за вказаний час дає "Критичний удар" (подвійний XP та бонусне золото).  
* **Нова шкала рівнів:** Тепер кожен наступний рівень вимагає більше досвіду (1000 \-\> 1400 \-\> 1800 і т.д.).  
* **Модальне вікно "Level Up":** Приємні візуальні ефекти та кумедні описи при досягненні нового рівня (аж до 100-го\!).  
* **Авторизація Google:** Безшовний вхід через Firebase Auth. Додано вікно вирішення конфліктів при розбіжностях між локальним і хмарним збереженням.  
* **Кнопка "Скасувати таймер":** Можливість миттєво відхилити активну сесію без штрафів та збереження часу (корисна іконка "Кошика").  
* **Модалка збереження сесії:** Вікно перед остаточною зупинкою таймера для швидкого редагування фактично відпрацьованого часу.

### **Змінено**

* Оновлено UI верхньої панелі: кнопки розділені на два ряди для кращого відображення на мобільних пристроях.  
* Візуальна зміна: Символ монети (🪙) тепер стоїть *перед* числовими значеннями для кращої естетики.

## **\[v1.4.0\] \- (Фаза 4: Мультимедіа та PWA)**

### **Додано**

* **PWA (Progressive Web App):** Вбудований генератор manifest.json для встановлення застосунку на смартфон чи ПК.  
* **Кастомні іконки:** Додані файли logo.png та logo-large.png для системних ярликів.  
* **Звукові ефекти (Web Audio API):** 8-бітні звуки покупок, заробітку монет, ударів меча, споживання зілля та фанфари для рівня.  
* **Генератор фонового патерну:** Ненав'язливий візерунок з RPG-емоджі (мечі, зілля, дракони), реалізований через CSS Mask Background.  
* **Випадкові події:** Міні-модалки з бонусами, які можуть з'явитися після 10+ хвилин фокусу з шансом 15%.  
* **Експорт даних:** Кнопка для завантаження детального звіту у форматі CSV.

## **\[v1.3.0\] \- (Фаза 3: Інтерфейс та Хмара)**

### **Додано**

* **Хмарна синхронізація:** Інтеграція Firebase Firestore. Дані автоматично зберігаються в хмарі кожні 5 секунд.  
* **Мультимовність (i18n):** Повна локалізація інтерфейсу Українською та Англійською мовами "на льоту".  
* **Світла / Темна тема:** Перемикач тем із збереженням налаштувань.  
* **Drag-and-Drop:** Можливість змінювати порядок світів, квестів та кроків мишкою чи пальцем.

## **\[v1.2.0\] \- (Фаза 2: Гейміфікація)**

### **Додано**

* **Крамниця Гільдії:** Зароблене золото тепер можна витратити на Зілля Здоров'я (+30 HP) або Зілля Фокусу (Подвійний XP на 30 хв).  
* **Стріки:** Комбо підряд проведених у застосунку днів дає бонусний множник досвіду. Втрата стріку карається відніманням HP.  
* **Штрафи за відволікання:** Зупинка таймера до завершення 5 хвилин забирає 10 HP ("шкода" від боса).  
* **Битви з Босами:** Перемикач "Бос" для проєкту. Створює спільну шкалу HP боса на основі кількості завдань.  
* **Глобальний XP:** Шкала досвіду тепер накопичувальна.  
* **Ручний ввід часу:** Модальне вікно для додавання часу, витраченого поза комп'ютером (з нарахуванням нагород).  
* **Архів:** Можливість завершити Світ та перемістити його у вкладку "Завершені".

## **\[v1.0.0\] \- (Фаза 1: MVP)**

### **Додано**

* **Базова структура:** Світи (Проєкти) \-\> Квести (Завдання) \-\> Кроки (Підзадачі).  
* **Таймер фокусу:** Відстеження витраченого часу на конкретну задачу з виводом на екран.  
* **Панель Героя:** Візуалізація Рівня, Здоров'я (HP), Досвіду (XP) та Золота (🪙).  
* **Лог пригод:** Стрічка останніх подій (нарахування досвіду та шкоди).  
* **Економіка:** Можливість встановити погодинну ставку (rate) для проєкту для генерації Золота.
